# 🔥 سیستم خودکار پایش قیمت طلا

## 📸 خروجی پروژه

![نمایش قیمت طلا](screenshot.png.png)

## 🤔 این پروژه چیکار میکنه؟

یک ربات پایتون هر روز صبح قیمت طلا رو از سایت tgju.org میگیره و توی فایل JSON ذخیره میکنه.  
یک صفحه PHP این قیمت رو خوند و به صورت زیبا به کاربر نشون میده.

## 🛠 تکنولوژی‌ها

- Python (Requests, JSON, datetime)
- PHP
- HTML/CSS/JS
- Windows Task Scheduler

## 💡 کاربرد برای کسب‌وکارها

- رصد خودکار قیمت رقبا
- مانیتورینگ محصولات فروشگاهی
- هشدار تغییر قیمت
- صرفه‌جویی ۶۰ ساعته در ماه
